# ☕ Coffee Store Data Analysis (Excel Project)

## 📊 Project Overview
This project focuses on analyzing coffee store sales data using Microsoft Excel. The goal is to extract meaningful insights related to sales performance, customer behavior, and product trends.
The dashboard provides a clear and interactive view of key business metrics to support data-driven decision-making.

## 🎯 Objectives
* Analyze sales performance across different locations
* Identify top-selling products
* Understand customer footfall patterns
* Track sales trends by day and month
* Evaluate category-wise revenue contribution

## 📁 Dataset Description
The dataset includes:
* Transaction details
* Product categories
* Sales amount
* Store locations
* Date and time of transactions

## 📌 Key Features of Dashboard
### 📅 Time-Based Analysis
* Sales trends by **day (Sunday–Saturday)**
* Monthly contribution to total sales
* Hourly transaction analysis

### 📍 Location Analysis
* Total sales by store location
* Footfall comparison across locations

### 🛍 Product Insights
* Top 5 selling products
* Category-wise revenue distribution

### 📦 Size Distribution
* Percentage contribution of different product sizes

## 📈 Tools & Techniques Used
* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning
* Data Visualization
* Dashboard Design

## 🔍 Key Insights
* Certain locations generate higher revenue and footfall
* Specific hours show peak customer activity
* A few products contribute significantly to total sales
* Monthly trends indicate seasonal variations

# 🚀 How to Use
1. Download the Excel file from the repository
2. Open in Microsoft Excel
3. Use filters (Month, Day) to interact with the dashboard
4. Explore different charts and insights

## 📌 Conclusion
This project demonstrates how Excel can be used as a powerful tool for data analysis and visualization. It highlights the importance of transforming raw data into actionable insights.
